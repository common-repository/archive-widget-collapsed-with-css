== Description ==

"Archive widget collapsed with CSS" is an archive widget that collapses and displays in CSS.

== Installation ==

1. Upload "Archive widget collapsed with CSS" folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You can set from the widget page.
